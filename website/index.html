Tools

Five tools, zero API keys

All data ships with the package. No external API calls, no rate limits, no authentication required.

get_matches
Query matches with flexible filters. Returns enriched data with team names, flag emojis, and venue details.
date date_range team group venue round status
get_teams
All 48 qualified nations with FIFA rankings, confederations, groups, and flag emojis.
group confederation is_host
get_groups
Full group details with teams, venues, and the complete match schedule for each group.
group
get_venues
16 host stadiums across USA, Mexico, and Canada. Capacity, coordinates, time zones, and notable events.
country city region
get_schedule
Tournament schedule organized by date. Every match day with kick-off times in UTC.
date_from date_to

Setup

Up and running in 2 minutes

Works with Claude Desktop, Cursor, Windsurf, and any MCP-compatible client.

1

Claude Desktop

Add to your claude_desktop_config.json:

{ "mcpServers": { "wc26": { "command": "npx", "args": ["-y", "wc26-mcp"] } } }
2

Claude Code

Add via the CLI:

claude mcp add wc26 -- npx -y wc26-mcp
3

Cursor

Add to your .cursor/mcp.json:

{ "mcpServers": { "wc26": { "command": "npx", "args": ["-y", "wc26-mcp"] } } }
4

Windsurf

Add to your ~/.codeium/windsurf/mcp_config.json:

{ "mcpServers": { "wc26": { "command": "npx", "args": ["-y", "wc26-mcp"] } } }
5

Ask away

Start asking about World Cup 2026:

"When does the USA play their first match?" "Show me all venues in Texas" "What teams are in Group C with Brazil?" "What's the schedule for the knockout rounds?"
