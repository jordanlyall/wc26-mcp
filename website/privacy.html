Privacy Policy

Last updated: February 11, 2026

Overview

WC26 MCP is an open-source MCP server that provides FIFA World Cup 2026 data. It is designed with privacy as a core principle.

Data Collection

WC26 MCP does not collect, store, transmit, or process any personal data. Zero. None.

Network Requests

The server makes no network requests at runtime. All data is statically bundled within the package. There are no API calls, no analytics, no telemetry, and no tracking of any kind.

Local Operation

WC26 MCP runs entirely on your local machine via stdio transport. No data leaves your computer through this server.

Third-Party Services

WC26 MCP does not integrate with any third-party services, advertising networks, or analytics platforms.

Open Source

The complete source code is publicly available at github.com/jordanlyall/wc26-mcp under the MIT license. You can verify all of the above by inspecting the code yourself.
